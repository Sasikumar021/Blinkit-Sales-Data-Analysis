# Blinkit-Sales-Data-Analysis

## 📊 Project Overview
This project analyzes Blinkit sales data and presents insights using an interactive Power BI dashboard.

## 🔧 Tools Used
- Power BI
- Excel

## 📌 Key Features
- Total Sales, Avg Sales, Item Count, Avg Rating KPIs
- Outlet-wise performance analysis
- Item type and fat content analysis
- Interactive filters (Outlet Size, Location, Item Type)

## 📈 Insights
- Tier 3 outlets generate highest revenue
- Grocery stores have higher item visibility
- Average ratings are consistent across categories
